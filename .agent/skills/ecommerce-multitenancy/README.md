# E-commerce Multitenancy Skill

Este es un AI Agent Skill para el proyecto de e-commerce. Guía al asistente sobre cómo mantener la arquitectura multitenant y el enrutamiento de Next.js.

## Instalación
Este skill ya está incrustado en el proyecto a través del directorio `.agent/skills/`.

## Características
- Define el flujo de enrutamiento del subdominio mediante `src/proxy.ts`.
- Mantiene la separación de las vistas de la "Landing page" global vs las "Tiendas de los inquilinos".
- Enfatiza el uso de pnpm y el idioma español.
